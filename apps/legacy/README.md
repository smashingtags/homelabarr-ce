# Legacy Apps

These templates are **unmaintained** and provided as-is. They were removed from the main catalog because the upstream projects are dead, abandoned, or duplicated by better alternatives.

**Want to use one?** Copy the YAML file into your `myapps/` folder and it will appear under "My Apps" in HomelabARR.

**No support provided.** Images may be outdated or unavailable. Use at your own risk.

## Why these were removed

| App | Reason |
|-----|--------|
| amd | Barely maintained, niche downloader |
| bazarr4k, prowlarr4k/hdr, radarr4k/hdr, sonarr4k/hdr, traktarr4k/hdr | Duplicate configs of base apps — power users can clone the base YAML |
| cloud9 | AWS killed open-source version — use Code Server or Coder |
| instaloader | CLI scraping tool, not a self-hosted app |
| koel | Dead music server, no updates since 2024 |
| pihole-cloudflared, pihole-unbound | Variants of Pi-hole — base pihole template kept |
| socket-proxy | Infrastructure plumbing, not a user-facing app |
| webgrabplus | Extremely niche EPG grabber |
| xteve, xteve-gluetun | Dead project — Threadfin is the maintained fork |
