# 🤖 Claude's Browser MCP Control Documentation

## Overview

This is Claude's personal documentation space for Browser MCP commands and navigation. I can store all my notes here for reference.
## Browser MCP Commands Available

Based on our setup with`browsermcp: npx @browsermcp/mcp@latest - ✅ Connected`:
### Navigation Commands

- **Go to [URL]**- Basic navigation
- **Navigate to [specific URL]**- Direct navigation
- **Go back/forward**- Browser history navigation
### Interaction Commands

- **Click [element/button]**- Click on page elements
- **Type [text] in [field]**- Form field interaction
- **Search for [query]**- Search functionality
- **Take screenshot**- Capture current page
### Content Commands

- **Get page content**- Extract text content
- **Get console logs**- Debug information
- **Get selected element**- Current DOM selection
